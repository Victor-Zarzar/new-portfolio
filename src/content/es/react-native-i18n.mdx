# Cómo internacionalizar tu aplicación React Native con Expo e i18n

## Introducción

i18n es una abreviación de "Internationalization" (internacionalización). Es el proceso de preparar una aplicación para adaptarse a diferentes idiomas y regiones — implica separar el contenido del código, localizar cadenas de texto y estructurar el proyecto para soportar múltiples locales.

## Paso a paso

### 1. Instalación de paquetes

Instala las dependencias necesarias con Expo:

```bash
npx expo install expo-localization react-i18next i18next
npx expo install @react-native-async-storage/async-storage
```

### 2. Configuración del i18n

Crea el archivo `./i18n/index.ts` con el siguiente contenido:

```ts
import AsyncStorage from "@react-native-async-storage/async-storage";
import * as Localization from "expo-localization";
import i18n from "i18next";
import { initReactI18next } from "react-i18next";
import type { Locale } from "@/shared/types/locale";
import translationEn from "./locales/en-US/translation.json";
import translationEs from "./locales/es-ES/translation.json";
import translationPt from "./locales/pt-BR/translation.json";

export const resources = {
  "pt-BR": { translation: translationPt },
  "en-US": { translation: translationEn },
  "es-ES": { translation: translationEs },
} as const;

const initI18n = async () => {
  let savedLanguage = (await AsyncStorage.getItem("language")) as Locale | null;

  if (!savedLanguage) {
    savedLanguage = Localization.getLocales()[0]?.languageTag as Locale;
  }

  i18n.use(initReactI18next).init({
    compatibilityJSON: "v4",
    resources,
    lng: savedLanguage,
    fallbackLng: "en-US",
    interpolation: {
      escapeValue: false,
    },
  });
};

initI18n();

export default i18n;
```

### 3. Importar en el Root Layout

En tu archivo raíz `_layout.tsx`, importa el i18n antes de cualquier otra cosa:

```tsx
import "@/shared/global.css";
import { ThemeProvider } from "@react-navigation/native";
import { PortalHost } from "@rn-primitives/portal";
import { Stack } from "expo-router";
import { StatusBar } from "expo-status-bar";
import { useColorScheme } from "nativewind";
import { NAV_THEME } from "@/shared/lib/theme";
import "../../i18n";

export { ErrorBoundary } from "expo-router";

export default function RootLayout() {
  const { colorScheme } = useColorScheme();

  return (
    <ThemeProvider value={NAV_THEME[colorScheme ?? "light"]}>
      <StatusBar style={colorScheme === "dark" ? "light" : "dark"} />
      <Stack />
      <PortalHost />
    </ThemeProvider>
  );
}
```

### 5. Usando las traducciones en los componentes

El hook `useTranslation` de `react-i18next` es todo lo que necesitas:

```tsx
import { useTranslation } from "react-i18next";

export default function Screen() {
  const { t } = useTranslation();

  return <Text>{t("home.welcome")}</Text>;
}
```

Ejemplo completo de componente de pantalla:

```tsx
import { Link, Stack } from "expo-router";
import {
  MoonStarIcon,
  SunIcon,
  ArrowRightIcon,
  BookOpenIcon,
} from "lucide-react-native";
import { useColorScheme } from "nativewind";
import * as React from "react";
import { View } from "react-native";
import { useTranslation } from "react-i18next";
import { Button } from "@/shared/components/ui/button";
import { Icon } from "@/shared/components/ui/icon";
import { Text } from "@/shared/components/ui/text";
import {
  Avatar,
  AvatarFallback,
  AvatarImage,
} from "@/shared/components/ui/avatar";

const THEME_ICONS = {
  light: SunIcon,

};

function ThemeToggle() {
  const { colorScheme, toggleColorScheme } = useColorScheme();
  return (
    <Button
      onPressIn={toggleColorScheme}
      size="icon"
      variant="ghost"
      className="ios:size-9 rounded-full web:mr-1"
    >
      <Icon as={THEME_ICONS[colorScheme ?? "light"]} className="size-5" />
    </Button>
  );
}

function HeaderRight() {
  return (
    <View className="flex-row items-center gap-1 web:mr-2">
      <ThemeToggle />
    </View>
  );
}

const SCREEN_OPTIONS = {
  title: "",
  headerTransparent: true,
  headerRight: () => <HeaderRight />,
};

export default function Screen() {
  const { t } = useTranslation();

  return (
    <>
      <Stack.Screen options={SCREEN_OPTIONS} />
      <View className="flex-1 items-center justify-center gap-12 p-8">
        <View className="items-center gap-3">
          <Avatar alt="Avatar">
            <AvatarImage
              source={{ uri: "https://github.com/mrzachnugent.png" }}
            />
            <AvatarFallback>
              <Text>RN</Text>
            </AvatarFallback>
          </Avatar>
          <Text className="text-2xl font-bold tracking-tight">
            {t("appName")}
          </Text>
          <Text className="text-center text-sm text-muted-foreground">
            {t(
              "appDescription",
              "Tu boilerplate está listo. Comienza a construir algo increíble.",
            )}
          </Text>
        </View>

        <View className="w-full max-w-sm gap-3">
          {[
            {
              step: "01",
              label: t("step1", "Edita app/index.tsx para comenzar"),
            },
            {
              step: "02",
              label: t("step2", "Guarda para ver tus cambios al instante"),
            },
            {
              step: "03",
              label: t("step3", "Construye tus funcionalidades sobre esto"),
            },
          ].map(({ step, label }) => (
            <View
              key={step}
              className="flex-row items-center gap-4 rounded-xl border border-border bg-card p-4"
            >
              <Text className="font-mono text-xs font-semibold text-muted-foreground">
                {step}
              </Text>
              <Text className="flex-1 text-sm">{label}</Text>
            </View>
          ))}

          <Link href="https://reactnativereusables.com" asChild>
            <Button className="w-full">
              <Icon as={BookOpenIcon} className="size-4" />

            </Button>

            href="https://github.com/founded-labs/react-native-reusables"
```

## Conclusión

¡Listo! Con estos pasos, tu aplicación React Native con Expo está completamente configurada para soportar múltiples idiomas con almacenamiento persistente y fallback automático al idioma del dispositivo. Simple, escalable y fácil de mantener.
